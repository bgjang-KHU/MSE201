# 리스트 자료형 (Lists)

---

앞서 우리는 여러 개의 문자를 순서대로 담는 **문자열(String)**에 대해 배웠습니다. 이번 섹션에서는 파이썬에서 가장 유연하고 강력한 순차적 데이터 구조인 **리스트(List)**를 알아봅시다.

## **📦 리스트란 무엇인가요?**
<br>
리스트는 대괄호 `[ ]`를 사용하여 정의하며, 그 안에 담기는 요소(Element)들은 쉼표(`,`)로 구분합니다. 리스트의 가장 큰 특징은 어떤 타입의 데이터든 가리지 않고 한데 담을 수 있다는 점입니다.

숫자들만 담기: `[1, 2, 3, 4, 5]`

문자열들만 담기: `["Apple", "Banana", "Cherry"]`

섞어서 담기: `[10, "Python", 3.14, True]`

### **🚀 TRY IT!**
{: .text-blue-200 }

다양한 형태의 리스트를 만들어보고, type() 함수로 자료형을 확인해 보세요.

```python
# 1. 비어있는 리스트 (나중에 데이터를 채울 때 사용합니다)
empty_list = []
# 2. 숫자와 문자열이 섞인 리스트
mixed_list = [1, "Hello", 3.14, False]
# 3. 리스트 안에 또 다른 리스트 넣기 (중첩 리스트)
nested_list = [1, 2, ["a", "b", "c"]]

print(type(mixed_list))
print(nested_list)
```

---

### **🔑 핵심 포인트: 리스트의 특징**  

**1. 순서(Order)가 있습니다**: 데이터가 들어간 순서대로 번호(인덱스)가 매겨집니다.    
**2. 수정 가능(Mutable)합니다**: 리스트는 생성 후에도 요소를 추가, 삭제, 변경할 수 있습니다. (이 점이 이후에 배울 **튜플**과 가장 큰 차이점입니다!)    
**3. 다양한 타입 허용**: 하나의 리스트에 정수, 실수, 문자열, 심지어 다른 리스트까지 동시에 담을 수 있습니다.

{: .highlight }
💡 리스트는 데이터가 늘어나거나 줄어들 수 있고, 그 안의 내용물을 언제든 바꿀 수 있기 때문에 프로그래밍에서 가장 빈번하게 사용되는 자료구조입니다.

### **🤔 Wait and Think!**
문자열에서 배웠던 **인덱싱(Indexing)**과 **슬라이싱(Slicing)**을 리스트에서도 그대로 쓸 수 있을까요?

```python
colors = ["Red", "Green", "Blue"]
print(colors[0])
```

## 리스트의 인덱싱과 슬라이싱
<br>
리스트도 문자열과 마찬가지로 데이터의 위치 번호인 **인덱스(Index)**를 가집니다. 문자열이 '글자' 하나하나를 가리켰다면, 리스트는 그 안에 담긴 '요소' 하나하나를 가리킵니다.

### **리스트 인덱싱 (Indexing)**

리스트의 각 요소는 0번부터 시작하는 번호를 가집니다.

{: .warning }
다시 한번 강조하면 Python에서 index는 1이 아닌 0부터 시작합니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
a = [10, 20, 30]

print(a[0])      # 첫 번째 요소: 10
print(a[0] + a[2]) # 10 + 30 = 40
print(a[-1])     # 마지막 요소: 30
```

**📦 중첩 리스트에서 데이터 꺼내기 (Deep Dive)**

리스트 안에는 또 다른 리스트가 들어갈 수 있습니다. 이때는 대괄호[ ]를 연속해서 사용하여 마치 주소의 '동-호수'를 찾아가듯 데이터를 꺼낼 수 있습니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
# 리스트 안에 리스트가 있는 경우
a = [1, 2, 3, ['a', 'b', 'c']]

print(a[3])      # ['a', 'b', 'c'] 출력
print(a[3][0])   # 네 번째 요소(리스트)의 첫 번째 요소인 'a' 출력
```

💡 삼중 리스트 도전!
아무리 깊게 숨겨져 있어도 차례대로 인덱스를 붙이면 찾아낼 수 있습니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
# 3층 구조의 리스트
a = [1, 2, ['a', 'b', ['Life', 'is']]]

# 'Life'를 꺼내려면?
# 1. a[2] -> ['a', 'b', ['Life', 'is']]
# 2. a[2][2] -> ['Life', 'is']
# 3. a[2][2][0] -> 'Life'
print(a[2][2][0])
```

### **리스트 슬라이싱 (Slicing)**

슬라이싱을 사용하면 리스트의 특정 범위를 통째로 잘라낼 수 있습니다. 형식은 문자열과 동일한 [시작:끝]이며, 마찬가지로 **끝 번호의 요소는 포함되지 않는다**는 점을 기억하세요!

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
a = [1, 2, 3, 4, 5]

print(a[0:2]) # 인덱스 0부터 1까지: [1, 2]
print(a[:3])  # 처음부터 인덱스 2까지: [1, 2, 3]
print(a[3:])  # 인덱스 3부터 끝까지: [4, 5]
```

> {: .result .fs-3 }
> > [1, 2]    
> > [1, 2, 3]  
> > [4, 5]


{: .warning }
> 인덱싱과 슬라이싱의 가장 큰 차이점 중 하나는 결과물의 **'모양'**입니다.  
> - 인덱싱(`a[0]`): 리스트 안의 값 자체를 꺼냅니다. (결과: `1`)  
> - 슬라이싱(`a[0:1]`): 값이 담긴 새로운 리스트를 만듭니다. (결과: `[1]`)  


### **🤔 Wait and Think!**
아래와 같이 중첩된 리스트에서 슬라이싱을 하면 결과가 어떻게 나올까요? 직접 실행하기 전에 머릿속으로 먼저 그려보세요.

```python
a = [1, 2, 3, ['a', 'b', 'c'], 4, 5]

print(a[2:5])
print(a[3][:2])
```

## 리스트의 연산과 길이 구하기
<br>
리스트도 문자열처럼 더하거나 곱할 수 있습니다. 이는 산술적인 계산이 아니라, 문자열의 연산과 같이 리스트라는 '데이터 주머니'를 합치거나 반복하는 과정입니다.

다음 예시를 통해 리스트를 더하고, 곱하고, 그 길이를 구해봅시다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
a = [1, 2, 3]
b = [4, 5, 6]

print(a + b)  # 리스트 더하기 
print(a * 3)  # 리스트 곱하기
print(len(a)) # 리스트 길이 구하기: 리스트 안에 담긴 요소의 개수를 반환
```
> {: .result .fs-3 }
> > [1, 2, 3, 4, 5, 6]    
> > [1, 2, 3, 1, 2, 3, 1, 2, 3]  
> > 3 

{: .warning }
> **🛑 주의: 초보자가 가장 많이 하는 실수!**   
> 리스트의 요소가 숫자라고 해서 + 연산이 숫자끼리의 덧셈이 되지는 않습니다.  
> 리스트 안의 숫자끼리 계산하고 싶다면, 나중에 배울 **반복문(for)**이나 NumPy 라이브러리를 사용해야 합니다.

### **🤔 Wait and Think!**
문자열과 숫자가 섞여 있을 때 len()은 어떻게 작동할까요? 그리고 중첩 리스트의 길이는 어떻게 계산될까요?

```python

# 문제 2: 중첩 리스트의 길이는?
```

## 리스트 값의 수정과 삭제
<br>
리스트는 문자열과 달리 생성된 후에도 자유롭게 요소를 바꾸거나 지울 수 있는 **'유연한 바구니'**입니다.

**1. 리스트 요소 수정하기**  
수정하고 싶은 위치의 인덱스를 지정하고, 새로운 값을 할당(=)하기만 하면 됩니다.  
**2. 리스트 요소 삭제하기 (`del` 함수)**  
`del` 함수를 사용하면 특정 인덱스에 있는 요소를 완전히 삭제할 수 있습니다. 삭제된 뒤에는 뒤에 있던 요소들이 앞으로 한 칸씩 당겨집니다. 

```python
# 리스트 요소 수정하기
a = [1, 2, 3]
a[2] = 4
print(a)

# 리스트 요소 삭제하기 
b = [1, 2, 3]
del b[1]
print(b)

# 슬라이싱을 이용해 여러 개 한번에 삭제하기
c = [1, 2, 3, 4, 5]
del c[2:]
print(c)
```

> {: .result .fs-3 }
> > [1, 2, 4]     
> > [1, 3]   
> > [1, 2]

### **🛑 주의: 문자열은 안 되는데 리스트는 된다?**

```python
# 1. 문자열 (수정 불가능 - Error!)
s = "ABC"
s[0] = "K"    # 에러 발생!

# 2. 리스트 (수정 가능 - OK!)
l = ["A", "B", "C"]
l[0] = "K"
print(l)      # ['K', 'B', 'C']
```

{: .highlight }
> 💡 **Mutable(가변) vs Immutable(불변)**  
> **리스트**는 내용을 바꿀 수 있는 **가변(Mutable) 자료형**입니다.  
> **문자열**, **튜플**은 내용을 바꿀 수 없는 **불변(Immutable) 자료형**입니다.

## 리스트 관련 메서드 (Method)
<br>
리스트는 데이터를 단순히 담는 것을 넘어, 스스로를 수정하고 정렬하는 다양한 '기술(메서드)'을 가지고 있습니다. 문자열과 마찬가지로 리스트 변수 이름 뒤에 '.'을 붙여 어러가지 기능을 사용할 수 있습니다.  

### **1. 요소 추가와 정렬**
가장 빈번하게 사용되는 기능입니다.  
**특히 `append`는 리스트의 꽃이라고 할 수 있습니다.**

- `append(x)`: 리스트의 맨 마지막에 요소 x를 추가합니다.
- `sort()`: 리스트의 요소를 순서대로(오름차순) 정렬합니다.
- `reverse()`: 현재 리스트의 순서를 거꾸로 뒤집습니다. (정렬이 아님에 주의!)

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
a = [1, 4, 3, 2]

# 요소 추가
a.append(5)
print(a)  # [1, 4, 3, 2, 5]

# 정렬하기
a.sort()
print(a)  # [1, 2, 3, 4, 5]

# 순서 뒤집기
a.reverse()
print(a)  # [5, 4, 3, 2, 1]
```

> {: .result .fs-3 }
> > [1, 4, 3, 2, 5]       
> > [1, 2, 3, 4, 5]   
> > [5, 4, 3, 2, 1]

### **🤔 Wait and Think!**
reverse()를 사용하지 않고 어떻게하면 리스트의 순서를 뒤집을 수 있을까요?  
Hint: 슬라이싱 기법을 활용하기!

<br>

### **2. 위치 지정 삽입과 값으로 삭제**
인덱스 번호를 활용하거나, 특정 값을 찾아서 지우는 정교한 기능들입니다.

- `insert(a, b)`: a번째 위치에 b를 삽입합니다.
- `remove(x)`: 리스트에서 **첫 번째로 나오는 x**를 삭제합니다.
- `pop()`: 리스트의 맨 마지막 요소를 꺼내고, 그 요소를 리스트에서 삭제합니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
a = [1, 2, 3, 1, 2, 3]

# 특정 위치에 삽입 (인덱스 0번에 10 넣기)
a.insert(0, 10)
print(a)  

# 특정 값 삭제 (첫 번째로 만나는 3만 지움)
a.remove(3)
print(a)  

# 마지막 요소 꺼내기
last_item = a.pop()
print(last_item) # 3
print(a)         
```

> {: .result .fs-3 }
> > [10, 1, 2, 3, 1, 2, 3]       
> > [10, 1, 2, 1, 2, 3]  
> > 3   
> > [10, 1, 2, 1, 2]

<br>

### **3. 정보 확인 및 확장**

- `index(x)`: 리스트에 x 값이 있으면 그 값의 인덱스 위치를 알려줍니다.
- `count(x)`: 리스트 안에 x가 몇 개 있는지 세어줍니다.
- `extend(list)`: 리스트 뒤에 또 다른 리스트를 더합니다. 

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
a = [1, 2, 3, 1]

print(a.index(3)) # 3의 위치인 2 출력
print(a.count(1)) # 1이 두 개이므로 2 출력

a.extend([4, 5])
print(a)          # [1, 2, 3, 1, 4, 5]       
```

> {: .result .fs-3 }
> > 2       
> > 2  
> > [1, 2, 3, 1, 4, 5] 

{: .warning }
> **🛑 주의: sort()와 reverse()의 특징**   
> 이 함수들은 리스트 자체를 직접 변경하지만, 결과값을 반환(return)하지는 않습니다.  

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
a = [3, 1, 2]
result = a.sort()
print(result) # None 출력 (주의!)
print(a)      # [1, 2, 3] (리스트 자체는 바뀌어 있음)       
```

{: .important }
>💡 **실전 팁 (TIP)**   
> 데이터를 추가할 때 `append`는 **값 하나**를 넣고, `extend`는 **리스트 덩어리**를 합친다는 차이점을 기억하세요.  
> 만약 `a.append([4, 5])`라고 하면 리스트 안에 리스트가 들어가는 중첩 구조가 됩니다.
