# 집합 자료형 (Sets)

---
집합은 수학의 집합 개념을 프로그래밍으로 구현한 자료형입니다. 가장 큰 특징은 **중복을 허용하지 않고, 순서가 없다는 점**입니다.

## **📦 집합 (Sets) 만들기**
<br>
중괄호 `{ }`를 사용하거나 `set()` 함수를 사용하여 만듭니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
s1 = {1, 2, 3, 2, 1}
print(s1)             # 결과: {1, 2, 3} (중복이 자동으로 제거됨)

s2 = set("Banana")
print(s2)             # 결과: {'B', 'a', 'n'} 
```

{: .warning } 
>비어있는 집합을 만들 때는 반드시 `s = set()`이라고 써야 합니다.  
>만약 `s = {}`**`라고 쓰면, 파이썬은 이것을 집합이 아니라 다음에 배울 **딕셔너리(Dictionary)**로 오해합니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
s1 = {1, 2, 3, 2, 1}
print(type(s1))

s2 = {}
print(type(s2))
```

> {: .result .fs-3 }
> > <class 'set'>  
> > <class 'dict'> 


### **🔑 집합의 2가지 핵심 특징**
1. **중복 불허 (No Duplicates)**: 어떤 값을 여러 번 넣어도 하나만 남습니다. 데이터에서 유니크한 값만 뽑아낼 때 매우 유용합니다.

2. **순서 없음 (Unordered)**: 리스트나 튜플과 달리 순서가 없어서 인덱싱(`s[0]`)을 할 수 없습니다.  
Tip: 인덱싱이 필요하다면 list()로 변환한 뒤 사용해야 합니다.

## **집합 연산 (수학적 활용)**
파이썬 집합은 교집합, 합집합, 차집합을 구하는 아주 간편한 방법을 제공합니다.

|**연산**|**기호**|**메서드**|**설명**|
|:-------|:-------|:-------|:-------|
|교집합| a & b | a.intersection(b) |공통된 요소만 추출|
|합집합| a \| b | a.union(b) |두 집합을 합침 (중복 제거)|
|차집합| a - b | a.difference(b) |한쪽 집합에만 있는 요소 추출|

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
s1 = {1, 2, 3}
s2 = {2, 3, 4}

# 기호 사용
print(s1 & s2)  # {2, 3}
print(s1 | s2)  # {1, 2, 3, 4}
print(s1 - s2)  # {1}

# 메서드 사용
print(s1.intersection(s2))  
print(s1.union(s2))  
print(s1.difference(s2))
```

## **🛠️ 집합 관련 주요 메서드**

- `add(x)`: 값 1개 추가
- `update([x, y])`: 여러 값 한꺼번에 추가
- `remove(x)`: 특정 값 제거 (값이 없으면 에러 발생)
- `discard(x)`: 특정 값 제거 (값이 없어도 에러가 나지 않음!)

### **1. 데이터 추가하기 (add vs update)**

- `add(x)`: 값 하나를 집어넣습니다.
- `update([x, y, ...])`: 리스트나 튜플로 여러 값을 추가합니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
s = {1, 2, 3}

# 값 1개 추가
s.add(4)
print(s)  # {1, 2, 3, 4}

# 여러 값 한꺼번에 추가 (중복된 값은 알아서 걸러집니다)
s.update([5, 6, 1]) 
print(s)  # {1, 2, 3, 4, 5, 6}
```

### **2. 데이터 삭제하기 (remove vs discard)**

- remove(x): 없는 값을 지우라고 하면 **에러(KeyError)**를 내며 프로그램을 멈춥니다.  
"내가 지우려는 값이 반드시 이 집합에 있어야만 해! 없으면 뭔가 큰일 난 거야!"라는 강한 확신이 있을 때 사용합니다.
- discard(x): 없는 값을 지우라고 해도 그냥 넘어갑니다.  
"있으면 지우고, 없으면 말지 뭐."라는 식으로 안정적인 실행이 우선일 때 사용합니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
s = {1, 2, 3}

# 정상 삭제
s.remove(1)
s.discard(2)
print(s)  # {3}

# 없는 값을 지우려고 할 때
s.discard(100)  # 에러 없이 조용히 통과 (안전!)
# s.remove(100)   # 🛑 여기서 프로그램이 멈추며 에러 발생!
```

### **🤔 Wait and Think!**
**"리스트에서 중복된 데이터를 한 번에 지우고 싶을 때 어떻게 할까요?"**  
가장 빠른 방법은 리스트를 set()으로 바꿨다가 다시 list()로 되돌리는 것입니다.

```python
my_list = [1, 2, 2, 3, 3, 3]
unique_list = list(set(my_list))
print(unique_list)  # [1, 2, 3]
```
