# NumPy: 고성능 과학 계산의 시작

---
**NumPy (Numerical Python)**는 Python 과학 계산의 가장 기본이 되는 모듈입니다. 내부적으로 C 언어로 구현되어 있어 리스트보다 훨씬 빠르고 효율적으로 대량의 데이터를 처리합니다.

## **1. 넘파이 불러오기 및 배열 생성**
<br>
NumPy를 사용하려면 먼저 모듈을 불러와야 합니다. 업계 표준 관례에 따라 `np`라는 별칭을 사용합니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
import numpy as np # 
# 1차원 배열 (Vector) 생성
x = np.array([7, 8, 9]) 
# 2차원 배열 (Matrix) 생성: 중첩 리스트 활용
y = np.array([[1, 2, 3], [4, 5, 6]]) 

print(x)
print(y)
print(type(x))
```
> {: .result .fs-3 }
> > [7 8 9]  
> > [[1 2 3]  
> >  [4 5 6]]    
> > <class 'numpy.ndarray'>

## **2. 배열의 속성: 모양(Shape)과 크기(Size)**
<br>
배열이 몇 행 몇 열인지 확인하는 것은 데이터 분석의 첫걸음입니다. 

{: .highlight }
몇 번을 강조해도 지나치지 않습니다. 데이터의 유형, 모양, 크기를 확인하는 것은 데이터 분석의 첫걸음입니다!!!

- `.shape`: 배열의 차원을 **튜플** `(행, 열)` 형태로 반환합니다. (속성이므로 괄호를 쓰지 않습니다.)
- `.size`: 배열에 담긴 전체 요소의 개수를 반환합니다.

### **🚀 TRY IT!**
{: .text-blue-200 }
위에서 만들었던 numpy array의 모양과 크기를 확인해봅시다.

```python
print(x.shape) 
print(y.shape) # (2, 3) -> 2행 3열 
print(y.size)  # 6 -> 총 6개의 데이터 
```

> {: .result .fs-3 }
> > (3,)   
> > (2, 3)    
> >  6    

### **🤔 Wait and Think!**
❓ 1차원 array의 shape은 왜 (3,) 일까요?

NumPy의 `.shape` 속성은 항상 **튜플 (Tuple)** 형태의 결과값을 반환합니다.

튜플 시간에 배웠던 내용을 떠올려보세요. Python에서 요소가 하나뿐인 튜플을 만들 때는 반드시 뒤에 쉼표를 붙여야 합니다.

- `(3)` : 그냥 괄호에 둘러싸인 정수 3
- `(3,)` : 요소가 3 하나인 튜플(Tuple)

**1차원 array는 행렬이 아닌 '선'의 개념이며, `(3,)`은 한 방향으로 테이터가 3개 존재함을 뜻합니다.**  

한쪽으로 길이가 같은 두 배열을 합칠 때도 1차원과 2차원을 합치려고 하면 이러한 이유로 에러가 나기도 합니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
import numpy as np
# 1차원 배열 (3,)
a = np.array([1, 2, 3]) 
# 2차원 배열 (1, 3)
b = np.array([[4, 5, 6]]) 

# ❌ 그냥 합치면 에러 발생 (ValueError: all the input arrays must have same number of dimensions)
np.concatenate((a, b), axis=0) 

# ✅ 해결: a를 (1, 3)으로 차원 확장 후 합치기
a_2d = a.reshape(1, -1)  # (3,) -> (1, 3)으로 변신!
result = np.concatenate((a_2d, b), axis=0)

print(result)
```

## **3. 규칙적인 배열 생성 (`arange` vs `linspace`)**
일일이 데이터를 입력하지 않고도 일정한 간격의 배열을 쉽게 만들 수 있습니다.

|**함수**|**특징**|**예시**|**결과**|
|:-------|:-------|:-------|:-------|
|`np.arange(a, b, x)`|시작(a)부터 끝(b, 미포함)까지 간격(x)으로 생성 |`np.arange(1, 7, 2)` | `[1, 3, 5]`|
|`np.linspace(a, b, n)`|시작(a)부터 끝(b, 포함)까지 개수(n)를 기준으로 생성|`np.linspace(0, 10, 3)` | `[ 0.  5. 10.]`|

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
import numpy as np # 
# 0.5부터 3 전까지 0.5 간격으로 생성
a = np.arange(0.5, 3, 0.5) 
# 0부터 18까지 딱 10개의 숫자를 균등하게 생성
b = np.linspace(0, 18, 10) 

print(a)
print(b)
```

## **4. 인덱싱과 슬라이싱 (2차원)**
2차원 배열에서는 [행, 열] 형식을 사용합니다. 콤마(,)를 기준으로 앞은 행, 뒤는 열을 나타냅니다.
인덱싱과 슬리이싱을 활용하면 행열의 요소들을 쉽게 꺼낼 수 있습니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
import numpy as np
x = [[1, 4, 3], [9, 2, 7]]
y = np.array(x) # 리스트를 numpy array로 변환	

print(y[0, 1])  # 0행 1열의 값
print(y[0])     # 0행의 전체 데이터
print(y[0, :])  # 0행의 전체 데이터  
print(y[:, 1])  # 모든 행의 1열 데이터  
```

> {: .result .fs-3 }
> > 4  
> > [1 4 3]  
> > [1 4 3]  
> > [4 2]

### **🔍 조건으로 데이터 뽑기 (Boolean Indexing)**
<br>
넘파이 배열은 특이하게도 숫자가 아닌 **'조건식'**을 대괄호 [ ] 안에 넣을 수 있습니다. 이것을 **불 인덱싱(Boolean Indexing)**이라고 부릅니다.

작동 원리는 다음과 같이 2단계로 이루어집니다.

**1. 조건에 맞는 '참/거짓' 확인**
배열에 비교 연산자(`>`)를 쓰면, 넘파이는 모든 요소를 하나씩 검사해서 `True` 또는 `False`로 가득 찬 배열을 만듭니다.

```python
temps = np.array([10, 500, 1200])
mask = temps > 1000 
print(mask)
```
> {: .result .fs-3 }  
> [False False  True]

**2. 'True'인 데이터만 뽑아내기**
이 지도를 다시 원래 배열의 인덱스로 넣어주면, `True`인 위치의 값만 살아남습니다.

```python
result = temps[mask] 
print(result)
```
> {: .result .fs-3 }  
> [1200]

**3. 한 방에 끝내기 (One-shot)**
실무에서는 굳이 마스크 변수를 따로 만들지 않고, 대괄호 안에 조건식을 바로 넣어 한 줄로 처리합니다.

```python
result = temps[temps > 1000] 
print(result) # [1200]
```

## **6. 특수 배열 및 연산**
초기값이 0이나 1인 배열을 미리 만들어두고 데이터를 채워 넣는 방식이 자주 쓰입니다.

- `np.zeros((행, 열))`: 모든 요소를 0으로 채움
- `np.ones((행, 열))`: 모든 요소를 1로 채움 
- `np.empty((행, 열))`: 메모리만 할당 (초기화 안 함) 
- `전치 행렬(.T)`: 행과 열을 서로 바꿈
- `전치 행렬(.traspose())`: 행과 열을 서로 바꿈

|**구분**|**np.zeros**|**np.empty**|
|:-------|:-------|:-------|
|역할| 모든 요소를 0으로 초기화 |메모리만 할당 (초기화 안함)|
|결과물|항상 모든 요소는 0|알 수 없는 ramdom values|
|속도| 상대적으로 조금 더 느림 | 가장 빠름 |
|안정성| 높음 (항상 0임을 보장) | 낮음 (어떤 값이 들어있을지 모름) |

### **🚀 TRY IT! 행열 미리 만들기**
{: .text-blue-200 }

```python
a = np.zeros((3,5))
b = np.ones((5,3))
c = np.empty(4)

print(a)  
print(b)
print(c)  
```

### **🚀 TRY IT! 행열의 연산**
{: .text-blue-200 }

```python
b = np.array([[1, 2], [3, 4]])
d = np.array([[3, 4], [5, 6]])

print(b + 2)  # 모든 요소에 2를 더함 (스칼라 연산)
print(b * 2)  # 모든 요소에 2를 곱함 (스칼라 연산)
print(b ** 2) # 모든 요소를 제곱함 (스칼라 연산)

print(b + d)  # 행열 간 연산
print(b * d)  # 행열 간 연산
```
