# 조건문 (If-Else Statements)
<br>
조건문 (Branching statements, If-Else statements, or If statements)은 **논리 표현식(Logical Expression)**의 결과가 참(`True`)인 경우에만 특정 코드 블록을 실행하는 구문입니다. 프로그램이 모든 코드를 순차적으로 실행하는 대신, 상황에 따라 실행 경로를 선택하도록 만드는 분기(Branching) 기능을 수행합니다.

### **🛠️ 기본 문법 (Simple if)**

Python은 `if` 키워드 뒤의 논리  표현식을 평가합니다.

```python
if 논리_표현식:
    실행문 (Code Block)
```

- `if` 키워드로 시작합니다.
- 참 (`True`)면 들여쓰기 된 실행문이 작동합니다.
- 거짓 (`False`)면 실행문을 건너뛰고 다음 코드로 진행합니다.
- 논리 표현식 뒤의 `:`와 if 문에 적용되는 실행문들의 들여쓰기 (Indentation)에 유의합니다.

### **🛠️ 확장 문법 (if-elif-else)**

고려해야 할 조건이 여러 개인 경우 `elif`(else if)를 추가하며, 모든 조건이 맞지 않을 때 실행할 코드는 `else` 블록에 작성합니다.

```python
if 조건_P:
    실행문_1  # P가 참일 때 실행
elif 조건_Q:
    실행문_2  # P가 거짓이고 Q가 참일 때 실행
elif 조건_R:
    실행문_3  # P, Q가 거짓이고 R이 참일 때 실행
else:
    실행문_4  # 위 모든 조건이 거짓일 때 실행
```

**💡 핵심 동작 원리**

1. 순차적 검사: 조건은 위에서부터 아래로 순서대로 검사합니다.

2. 단일 블록 실행: 하나의 조건이 참이 되어 해당 코드 블록이 실행되면, 파이썬은 나머지 조건들을 검사하지 않고 조건문 전체를 즉시 종료합니다.

3. 구성 규칙:
- `if`문은 처음에 반드시 한 번 사용해야 합니다.
- `elif`는 개수에 제한이 없으며, 생략 가능합니다.
- `else`는 마지막에 단 한 번만 올 수 있으며, 생략 가능합니다.

{: .important }
**들여쓰기(Indentation)의 중요성**
파이썬은 들여쓰기를 통해 어떤 코드가 조건문에 포함되는지를 판단합니다. 동일한 조건 아래 실행될 문장들은 반드시 동일한 깊이로 들여쓰기 되어야 함을 유의해야 합니다.
