# 변수와 할당 (Variables and Assignment)

---

프로그래밍을 할 때 정보를 저장해두고 필요할 때 꺼내 쓰는 것은 매우 유용합니다. 이때 정보를 담는 그릇 역할을 하는 것이 바로 **변수(Variable)**입니다.
다시 말해, 데이터를 저장하는 가장 기본적인 방법은 **변수(Variable)**를 사용하는 것입니다.

* **변수(Variable)**: 특정 정보와 연결된 문자나 숫자의 조합(이름) 말합니다.
* **할당 연산자 (=)**: Python에서 등호(=)를 사용하여 변수에 값을 집어넣습니다. 이를 '할당(Assignment) 연산자'라고 부릅니다.

```python
x=1
print(x)
```

> {: .result }
1 


위 코드는 "x는 1과 같다"는 수학적 의미가 아니라, **숫자 1을 x라는 이름의 메모리 공간에 저장**하겠다는 의미입니다.

## 변수 명명 규칙

1. 영문자, 숫자, 언더스코어(`_`)만 사용할 수 있다.
2. 숫자로 시작할 수 없다.
3. Python에서 특별한 의미로 사용되는 예약어는 사용할 수 없다.

   `False` `None` `True` `and` `def` `elif` `if` `import` `or` `return` `while` 등
4. 대소문자를 구분한다.

{: .warning }
Python에서는 이미 저장된 변수나 함수를 덮어쓸 수 있습니다. 예를 들어, `help = 2`라는 명령을 실행하면 `help`라는 이름의 변수에 `2`라는 값이 저장됩니다. 이 할당이 이루어진 후에는 `help`가 기존의 도움말 함수가 아니라 숫자 2처럼 동작하게 됩니다. 따라서 변수 이름을 지을 때는 파이썬의 내장 함수나 기본값과 같은 이름을 사용하지 않도록 항상 주의해야 합니다.

## 변수명 권장 사항
1. 의미가 명확한 이름을 사용한다.
2. 단어 사이에는 언더스코어(`_`)를 권장한다.
3. 너무 짧거나 긴 이름은 피한다.

---
### **🚀 TRY IT!**
{: .text-blue-200 }

```python
x=1
print(x)
x=x+1
print(x)
```
> {: .result }
> > 1
> >
> > 2 

수학적으로 `x = x + 1`이라는 문장은 그 어떤 `x` 값에 대해서도 성립할 수 없는(해가 없는) 모순된 식입니다. 하지만 프로그래밍의 세계에서는 이 문장은 완벽한 논리를 가집니다.

만약 우리가 변수 `x`의 초기값을 `1`로 설정했다면, `x = x + 1`은 다음과 같은 순서로 해석됩니다.

1. 계산: 현재 `x`의 값인 `1`에 `1`을 더해 `2`를 만듭니다.  
2. 할당: 계산된 결과값(`2`)을 다시 변수 `x`에 집어넣습니다.

결과적으로 이 연산은 `x`에 저장되어 있던 이전 값을 지우고 새로운 값으로 덮어쓰는(Overwrite) 과정입니다.

수학에서 변수는 대개 **미지수**를 의미하지만, 프로그래밍에서 변수는 특정한 타입(유형)을 가진 값과 연결됩니다.

변수에는 다양한 데이터 타입을 할당할 수 있습니다. **데이터 타입(Data type)**이란 변수에 저장되는 정보의 종류를 분류한 것입니다. 여러분이 사용할 기본적인 데이터 타입은 **불(boolean), 정수(int), 실수(float), 문자열(string), 리스트(list), 튜플(tuple), 딕셔너리(dictionary), 집합(set)**입니다. 


{: .note }
"수학의 *x*는 답을 구해야 하는 대상이지만, 파이썬의 `x`는 데이터를 담고 있는 이름표와 같다"
