# 불 자료형 (Boolean)

---

앞서 1장에서 우리는 `True`와 `False`를 다루는 논리 연산자 (Logical operators)와 불 자료형에 대해 간단히 배웠습니다. 이번 챕터에서는 개념을 조금 더 보충하고, 앞으로 배우게 될 `if`문, `while`문 등에서 어떻게 사용될 수 있는지 감을 익혀봅시다!

## **🔑 자료형의 참과 거짓**
<br>
앞서 배운 것과 같이 Python의 모든 자료형(문자열, 리스트, 숫자 등)은 그 자체로 **참(True)** 혹은 **거짓(False)**의 성질을 가지고 있습니다.  
참과 거짓을 나누는 기준은 단순합니다. 자료가 **"비어 있거나 0인가?"**를 확인하면 됩니다.

|**데이터 타입**|**참(True)**|**거짓(False)**|
|:-------|:-------|:-------|:-------|
|문자열|`"python"`,`"  "` (공백 포함) | `""` (빈 문자열) |
|리스트|`[1,2,3]` | `[]` (빈 리스트) |
|튜플|`(1,2,3)` | `()` (빈 튜플) |
|딕셔너리|`{"a": 1}` | `{}` (빈 튜플) |
|숫자형|`0`이 아닌 모든 숫자 | `0` |
|기타| - | `None` (값이 없음)|

---
## **🚀 실전 활용: 조건문과 반복문**
<br>
데이터의 참/거짓 성질은 나중에 배울 `if`나 `while` 문에서 코드를 아주 짧고 강력하게 만들어 줍니다.

### **1. 리스트가 비어있는지 확인하기 (if)**
"리스트의 길이가 0인가?"라고 굳이 묻지 않아도, 리스트 객체 그 자체를 조건으로 쓸 수 있습니다.

```python
data = []

if data:
    print("데이터가 존재합니다.")
else:
    print("데이터가 없습니다.") # 결과: 데이터가 없습니다.
```

### **2. 리스트가 비워질 때까지 실행하기 (while)**
리스트에서 요소를 하나씩 꺼내다가, 리스트가 완전히 비워지면(거짓이 되면) 자동으로 멈추는 아주 파이썬스러운(Pythonic) 코드입니다.

```python
stack = [10, 20, 30]

# stack이 참인 동안(즉, 요소가 남아있는 동안) 반복
while stack:
    item = stack.pop()
    print(f"꺼낸 값: {item}, 남은 바구니: {stack}")

print("작업 완료!")
```

## **🛠️ bool() 함수로 직접 확인하기**
특정 데이터가 참인지 거짓인지 헷갈릴 때는 bool() 함수를 사용하여 판별해 볼 수 있습니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
print(bool('python')) # True
print(bool(''))       # False
print(bool(0))        # False
print(bool(3.14))     # True
```
