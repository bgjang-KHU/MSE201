# 집합 자료형 (Sets)

---
집합은 수학의 집합 개념을 프로그래밍으로 구현한 자료형입니다. 가장 큰 특징은 **중복을 허용하지 않고, 순서가 없다는 점**입니다.

## **📦 집합 (Sets) 만들기**
<br>
중괄호 { }를 사용하거나 set() 함수를 사용하여 만듭니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
s1 = {1, 2, 3, 2, 1}
print(s1)             # 결과: {1, 2, 3} (중복이 자동으로 제거됨)

s2 = set("Banana")
print(s2)             # 결과: {'B', 'a', 'n'} (순서가 뒤죽박죽이며 중복 제거)
```

### **🔑 집합의 2가지 핵심 특징**
1. **중복 불허 (No Duplicates)**: 어떤 값을 여러 번 넣어도 하나만 남습니다. 데이터에서 유니크한 값만 뽑아낼 때 매우 유용합니다.

2. **순서 없음 (Unordered)**: 리스트나 튜플과 달리 순서가 없어서 인덱싱(`s[0]`)을 할 수 없습니다.  
Tip: 인덱싱이 필요하다면 list()로 변환한 뒤 사용해야 합니다.
