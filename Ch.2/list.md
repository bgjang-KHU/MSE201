# 리스트 자료형 (List)

---

앞서 우리는 여러 개의 문자를 순서대로 담는 **문자열(String)**에 대해 배웠습니다. 이번 섹션에서는 파이썬에서 가장 유연하고 강력한 순차적 데이터 구조인 **리스트(List)**를 알아봅시다.

### **📦 리스트란 무엇인가요?**
<br>
리스트는 대괄호 `[ ]`를 사용하여 정의하며, 그 안에 담기는 요소(Element)들은 쉼표(`,`)로 구분합니다. 리스트의 가장 큰 특징은 어떤 타입의 데이터든 가리지 않고 한데 담을 수 있다는 점입니다.

숫자들만 담기: `[1, 2, 3, 4, 5]`

문자열들만 담기: `["Apple", "Banana", "Cherry"]`

섞어서 담기: `[10, "Python", 3.14, True]`

### **🚀 TRY IT!**
{: .text-blue-200 }

다양한 형태의 리스트를 만들어보고, type() 함수로 자료형을 확인해 보세요.

```python
# 1. 비어있는 리스트 (나중에 데이터를 채울 때 사용합니다)
empty_list = []

# 2. 숫자와 문자열이 섞인 리스트
mixed_list = [1, "Hello", 3.14, False]

# 3. 리스트 안에 또 다른 리스트 넣기 (중첩 리스트)
nested_list = [1, 2, ["a", "b", "c"]]

print(type(mixed_list))
print(nested_list)
```

---

### **🔑 핵심 포인트: 리스트의 특징**  

**1. 순서(Order)가 있습니다**: 데이터가 들어간 순서대로 번호(인덱스)가 매겨집니다.    
**2. 수정 가능(Mutable)합니다**: 리스트는 생성 후에도 요소를 추가, 삭제, 변경할 수 있습니다. (이 점이 이후에 배울 **튜플**과 가장 큰 차이점입니다!)    
**3. 다양한 타입 허용**: 하나의 리스트에 정수, 실수, 문자열, 심지어 다른 리스트까지 동시에 담을 수 있습니다.

{: .highlight }
💡 리스트는 데이터가 늘어나거나 줄어들 수 있고, 그 안의 내용물을 언제든 바꿀 수 있기 때문에 프로그래밍에서 가장 빈번하게 사용되는 자료구조입니다.

### **🤔 Wait and Think!**
문자열에서 배웠던 **인덱싱(Indexing)**과 **슬라이싱(Slicing)**을 리스트에서도 그대로 쓸 수 있을까요?

```python
colors = ["Red", "Green", "Blue"]
print(colors[0])
```

## 리스트의 인덱싱과 슬라이싱
<br>
리스트도 문자열과 마찬가지로 데이터의 위치 번호인 **인덱스(Index)**를 가집니다. 문자열이 '글자' 하나하나를 가리켰다면, 리스트는 그 안에 담긴 '요소' 하나하나를 가리킵니다.

### 리스트 인덱싱 (Indexing)
리스트의 각 요소는 0번부터 시작하는 번호를 가집니다.

{: .warning }
다시 한번 강조하면 Python에서 index는 1이 아닌 0부터 시작합니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
a = [10, 20, 30]

print(a[0])      # 첫 번째 요소: 10
print(a[0] + a[2]) # 10 + 30 = 40
print(a[-1])     # 마지막 요소: 30
```
