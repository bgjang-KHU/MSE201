# 숫자형 

---

|  항목   |   예시            |
|:-------|:------------------|
|Integer (int, 정수)              | 123, -345, 0        |
|Floating-point (float, 실수)     | 123.45, 1.0, 3.4e10 |
|Complex number (complex, 복소수) | 1+2j, -3J            |

### 정수형

정수형(integer)은 말 그대로 정수를 뜻하는 자료형이다. 

```python
a=123
print(a)
print(type(a))
```
> {: .result }
> > 123
> > <class 'int'>
