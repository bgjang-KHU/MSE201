# 반복문1 (For Loops)
<br>
For 문은 리스트, 튜플, 문자열과 같은 **시퀀스(Sequence)** 내의 모든 요소에 대해 특정 코드 블록을 되풀이하여 실행하는 구문입니다. 참고 자료에서는 이를 **확정적 반복문(Definite loops)** 이라고도 부르는데, 시퀀스의 시작과 끝이 미리 정해져 있기 때문입니다.

## **🛠️ 기본 문법**

파이썬의 for 문은 문장 구조가 매우 직관적입니다.

```python
for 변수 in 시퀀스(리스트, 튜플, 문자열 등):
    수행할 문장 1
    수행할 문장 2
```

- **`for` 키워드**: 반복의 시작을 알립니다.
- **루프 변수 (Looping Variable)**: 시퀀스의 요소를 차례대로 하나씩 담는 바구니 역할을 합니다.
- **`in` 키워드**: 뒤에 올 시퀀스 데이터에서 값을 가져오겠다는 뜻입니다.
- **시퀀스**: 반복할 대상(리스트, 문자열 등)입니다. 첫 번째 요소부터 마지막까지 차례로 변수에 대입됩니다.
- **콜론(:)과 들여쓰기**: if 문과 마찬가지로 반복 실행될 문장들은 반드시 들여쓰기 되어야 합니다.

**🔄 동작 원리**

1. 시퀀스의 첫 번째 요소를 루프 변수에 대입합니다.

2. 들여쓰기 된 코드 블록을 실행합니다.

3. 시퀀스의 다음 요소가 있으면 다시 변수에 대입하고 코드 블록을 실행합니다.

4. 시퀀스의 마지막 요소까지 처리가 끝나면 반복문을 종료하고 다음 코드로 넘어갑니다.

---

## **기초 예시**

```python
# 리스트 순회하기
fruits = ["apple", "banana", "cherry"]

for x in fruits:
    print(f"내가 좋아하는 과일은 {x}입니다.")

# 문자열 순회하기 (문자 하나씩 출력)
for char in "PYTHON":
    print(char)
```
