# 반복문1 (For Loops)
<br>
For 문은 리스트, 튜플, 문자열과 같은 **시퀀스(Sequence)** 내의 모든 요소에 대해 특정 코드 블록을 되풀이하여 실행하는 구문입니다. 참고 자료에서는 이를 **확정적 반복문(Definite loops)** 이라고도 부르는데, 시퀀스의 시작과 끝이 미리 정해져 있기 때문입니다.

## **🛠️ 기본 문법**

파이썬의 for 문은 문장 구조가 매우 직관적입니다.

```python
for 변수 in 시퀀스(리스트, 튜플, 문자열 등):
    수행할 문장 1
    수행할 문장 2
```

- **`for` 키워드**: 반복의 시작을 알립니다.
- **루프 변수 (Looping Variable)**: 시퀀스의 요소를 차례대로 하나씩 담는 바구니 역할을 합니다.
- **`in` 키워드**: 뒤에 올 시퀀스 데이터에서 값을 가져오겠다는 뜻입니다.
- **시퀀스**: 반복할 대상(리스트, 문자열 등)입니다. 첫 번째 요소부터 마지막까지 차례로 변수에 대입됩니다.
- **콜론(:)과 들여쓰기**: if 문과 마찬가지로 반복 실행될 문장들은 반드시 들여쓰기 되어야 합니다.

**🔄 동작 원리**

1. 시퀀스의 첫 번째 요소를 루프 변수에 대입합니다.
2. 들여쓰기 된 코드 블록을 실행합니다.
3. 시퀀스의 다음 요소가 있으면 다시 변수에 대입하고 코드 블록을 실행합니다.
4. 시퀀스의 마지막 요소까지 처리가 끝나면 반복문을 종료하고 다음 코드로 넘어갑니다.

## **🛠️ 연속된 숫자 생성기: `range()` 함수**

특정 횟수만큼 반복하거나 일정한 규칙의 숫자 리스트가 필요할 때 사용합니다. Python의 `range()`는 메모리를 효율적으로 쓰기 위해 숫자를 미리 다 만들어두지 않고, 필요할 때마다 하나씩 꺼내주는 함수이며, **`for` 문에서 시퀀스를 정해줄 때 가장 흔하게 사용됩니다**.


- `range(stop)` : 0부터 stop 미만까지 1씩 증가
- `range(start, stop)` : start부터 stop 미만까지 1씩 증가
- `range(start, stop, step)` : start부터 stop 미만까지 step만큼 증가

```python
for i in range(0,5):
    print(i)
```
> {: .result .fs-3 }
> 0  
> 1  
> 2  
> 3  
> 4

### **💡range vs. arange vs. linspace**

|**구분**|**`range()`**|**`np.arange()`**|**`np.arange()`**|
|:-------|:-------|:-------|:-------|
|특징1 | Python 기본 제공 | Numpy 라이브러리 |Numpy 라이브러리|
|특징2 | **정수(int)**만 가능 | 정수, **실수(float)** 모두 가능| 정수, **실수(float)** 모두 가능|
|특징3| 반복 횟수 제어용 | 시작/끝/**간격** 지정 |시작/끝/**개수** 지정|
|예시| `range(0, 5)` | `np.arange(0, 0.5, 0.1)` | `np.linspace(0, 1, 10)`|

### **🚀 TRY IT!**
{: .text-blue-200 }

간단한 반복문을 사용해 1에서 3까지의 모든 정수의 합을 구해봅시다.

```python
sum=0
for i in range(1,4):
    sum+=i
print(f'총합은 {sum}입니다')
```
> {: .result .fs-3 }
> 총합은 6입니다  

### **🤔 Wait and Think!**
위의 예시 코드를 실행할 때 어떤 일이 일어나고 있는 것일까요?

1. 우선 `range(1,4)` 함수는 1로 시작해 3에서 끝나는 리스트를 만듭니다.
2. 반복 1회차 
    - 변수 `i`에 첫번재 값이 1이 할당됩니다.
    - n=n+i 계산이 수행됩니다 (0+1=1)
    - 여기에서 n의 값은 1입니다.
3. 반복 2회차 
    - 변수 `i`에 두번재 값이 2이 할당됩니다.
    - n=n+i 계산이 수행됩니다 (1+2=3)
    - 여기에서 n의 값은 3입니다.
4. 반복 3회차 
    - 변수 `i`에 첫번재 값이 3이 할당됩니다.
    - n=n+i 계산이 수행됩니다 (3+3=6)
    - 여기에서 n의 값은 6입니다.
5. 리스트에 더 이상 할당할 값이 없으므로, `for`문이 종료됩니다. 최종적인 `n`의 값은 6이 됩니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

딕셔너리 자료형에서는 다음과 같이 `for` 문을 사용할 수 있습니다.

```python
dict_a = {"One":1, "Two":2, "Three":3}

for key in dict_a.keys():
    print(key, dict_a[key])
```
> {: .result .fs-3 }  
> One 1  
> Two 2  
> Three 3

위 예시에서는 우선 `key()` 메서드를 사용해 키 (key)들을 얻어오고, 키 (key)를 사용해서 해당하는 값 (value)에 접근했습니다.
하지만 아래와 같이 한번에 key와 value를 얻는 것도 가능합니다.

```python
dict_a = {"One":1, "Two":2, "Three":3}

for key, value in dict_a.items():
    print(key, value)
```

위 예시에서는 서로 다른 두 개의 루프 변수를 동시에 할당한 것입니다. 비슷한 방식으로 만약 두 개의 같은 길이를 갖는 리스트 (list)가 있다면 다음과 같이 `zip` 함수를 사용할 수 있습니다.

```python
a = ["One", "Two", "Three"]
b = [1, 2, 3]

for i, j in zip(a, b):
    print(i, j)
```

## **루프 제어문 (break, continue, pass)**

반복문이 실행되는 도중, 특정 조건에 따라 반복을 멈추거나 다음 순서로 넘어가야 할 때 사용하는 키워드들입니다.

### **🚫 break: "여기서 당장 멈춰!"**
`break` 키워드는 자신을 감싸고 있는 가장 가까운 반복문을 즉시 종료하고 빠져나옵니다.

🤔 언제 쓸까요? 무한 루프에서 특정 목표치에 도달했거나, 에러가 발생하여 더 이상 계산이 불가능할 때 사용합니다.

```python
for i in range(1, 11):
    if i == 5:
        break  # i가 5가 되는 순간 반복문을 완전히 끝냅니다.
    print(i)

# 출력 결과: 1, 2, 3, 4 (5부터는 출력되지 않음)
```

### **⏭️ continue: "이번 판은 건너뛰자!"**
`continue` 키워드는 현재 반복 주기의 남은 코드를 모두 무시하고, 시퀀스의 다음 요소로 즉시 넘어가 반복을 계속합니다.

🤔 언제 쓸까요? 특정 데이터(예: 음수 값이나 결측치)를 제외하고 계산을 이어가고 싶을 때 유용합니다.

```python
for i in range(1, 6):
    if i == 3:
        continue  # i가 3일 때는 아래 print를 무시하고 4로 넘어갑니다.
    print(i)

# 출력 결과: 1, 2, 4, 5 (3만 쏙 빠짐)
```

### **😶 pass: "일단 자리만 지키고 있어""**

`pass`는 아무런 동작도 하지 않는 키워드입니다. 문법적으로는 문장이 필요하지만, 실제로는 아무 일도 하고 싶지 않을 때 사용합니다.

🤔 언제 쓸까요? 나중에 코드를 작성할 부분을 미리 만들어 두거나(Placeholder), 조건문에서 아무 처리도 하지 않고 넘어가야 할 때 씁니다.

```python
for i in range(1, 4):
    if i == 2:
        pass  # 나중에 여기에 특별한 로직을 넣을 예정!
    print(i)

# 출력 결과: 1, 2, 3 (아무 변화 없음)
```
