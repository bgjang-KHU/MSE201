# 반복문1 (For Loops)
<br>
For 문은 리스트, 튜플, 문자열과 같은 **시퀀스(Sequence)** 내의 모든 요소에 대해 특정 코드 블록을 되풀이하여 실행하는 구문입니다. 참고 자료에서는 이를 **확정적 반복문(Definite loops)** 이라고도 부르는데, 시퀀스의 시작과 끝이 미리 정해져 있기 때문입니다.

## **🛠️ 기본 문법**

파이썬의 for 문은 문장 구조가 매우 직관적입니다.

```python
for 변수 in 시퀀스(리스트, 튜플, 문자열 등):
    수행할 문장 1
    수행할 문장 2
```

- **`for` 키워드**: 반복의 시작을 알립니다.
- **루프 변수 (Looping Variable)**: 시퀀스의 요소를 차례대로 하나씩 담는 바구니 역할을 합니다.
- **`in` 키워드**: 뒤에 올 시퀀스 데이터에서 값을 가져오겠다는 뜻입니다.
- **시퀀스**: 반복할 대상(리스트, 문자열 등)입니다. 첫 번째 요소부터 마지막까지 차례로 변수에 대입됩니다.
- **콜론(:)과 들여쓰기**: if 문과 마찬가지로 반복 실행될 문장들은 반드시 들여쓰기 되어야 합니다.

**🔄 동작 원리**

1. 시퀀스의 첫 번째 요소를 루프 변수에 대입합니다.
2. 들여쓰기 된 코드 블록을 실행합니다.
3. 시퀀스의 다음 요소가 있으면 다시 변수에 대입하고 코드 블록을 실행합니다.
4. 시퀀스의 마지막 요소까지 처리가 끝나면 반복문을 종료하고 다음 코드로 넘어갑니다.

## **🛠️ 연속된 숫자 생성기: `range()` 함수**

특정 횟수만큼 반복하거나 일정한 규칙의 숫자 리스트가 필요할 때 사용합니다. Python의 `range()`는 메모리를 효율적으로 쓰기 위해 숫자를 미리 다 만들어두지 않고, 필요할 때마다 하나씩 꺼내주는 함수이며, **`for` 문에서 시퀀스를 정해줄 때 가장 흔하게 사용됩니다**.


- `range(stop)` : 0부터 stop 미만까지 1씩 증가
- `range(start, stop)` : start부터 stop 미만까지 1씩 증가
- `range(start, stop, step)` : start부터 stop 미만까지 step만큼 증가

```python
for i in range(0,5):
    print(i)
```
> {: .result .fs-3 }
> 0  
> 1  
> 2  
> 3  
> 4

### **💡range vs. arange vs. linspace**

|**구분**|**`range()`**|**`np.arange()`**|**`np.arange()`**|
|:-------|:-------|:-------|:-------|
|특징1 | Python 기본 제공 | Numpy 라이브러리 |Numpy 라이브러리|
|특징2 | **정수(int)**만 가능 | 정수, **실수(float)** 모두 가능| 정수, **실수(float)** 모두 가능|
|특징3| 반복 횟수 제어용 | 시작/끝/**간격** 지정 |시작/끝/**개수** 지정|
|예시| `range(0, 5)` | `np.arange(0, 0.5, 0.1)` | `np.linspace(0, 1, 10)`|

### **🚀 TRY IT!**
{: .text-blue-200 }

간단한 반복문을 사용해 1에서 3까지의 모든 정수의 합을 구해보자.

```python
sum=0
for i in range(1,4):
    sum+=i
print(f'총합은 {sum}입니다')
```
> {: .result .fs-3 }
> 총합은 6입니다  

### **🤔 Wait and Think!**
위의 예시 코드를 실행할 때 어떤 일이 일어나고 있는 것일까요?

1. 우선 `range(1,4)` 함수는 1로 시작해 3에서 끝나는 리스트를 만듭니다.
2. 반복 1회차 
    - 변수 `i`에 첫번재 값이 1이 할당됩니다.
    - n=n+i 계산이 수행됩니다 (0+1=1)
    - 여기에서 n의 값은 1입니다.
3. 반복 2회차 
    - 변수 `i`에 두번재 값이 2이 할당됩니다.
    - n=n+i 계산이 수행됩니다 (1+2=3)
    - 여기에서 n의 값은 3입니다.
4. 반복 3회차 
    - 변수 `i`에 첫번재 값이 3이 할당됩니다.
    - n=n+i 계산이 수행됩니다 (3+3=6)
    - 여기에서 n의 값은 6입니다.
5. 리스트에 더 이상 할당할 값이 없으므로, `for`문이 종료됩니다. 최종적인 `n`의 값은 6이 됩니다.

반복문을 사용해서 리스트 안에 있는 숫자의 합을 구해봅시다.

```python
sum = 0

```

`range()`함수와 `len()`함수를 사용하여 같은 동작을 하는 코드를 작성해봅시다.

```python
sum = 0
a = [2, 3, 1, 3, 3]
for i in range(0, len(a)):
    sum += a[i] 
print(sum)
```
