# 반복문2 (While Loops)
<br>
`while` 문은 주어진 논리 표현식(Logical Expression)이 참(True)인 동안 코드 블록을 무한히 반복합니다. 반복 횟수가 미리 정해져 있지 않은 경우가 많아 **부정적 반복문(Indefinite loop)**이라고도 불립니다.

## **🛠️ 기본 문법**

```python
while 조건문:
    수행할 문장 1
    수행할 문장 2
```

- **`while` 키워드**: 반복의 시작을 알립니다.
- **조건문**: 반복을 계속할지 결정하는 논리 표현식입니다. 이 식의 결과가 `True`이면 반복하고, `False`가 되면 즉시 멈춥니다.
- **콜론(:)과 들여쓰기**: `while` 문에 속한 실행문들은 반드시 동일한 간격으로 들여쓰기 되어야 합니다. 들여쓰기가 끝나는 지점이 반복문이 끝나는 지점입니다.

**🔄 동작 원리**

1. 먼저 `while` 뒤의 조건문이 참인지 거짓인지 검사합니다.
2. 조건이 **참(`True`)**이라면 들여쓰기 된 코드 블록을 실행합니다.
3. 록 내의 모든 문장을 실행한 후, 다시 위로 돌아가 **1번(조건 판단)**부터 반복합니다.
4. 조건이 **거짓(`False`)**이 되면 반복문을 빠져나와 다음 코드로 진행합니다.
