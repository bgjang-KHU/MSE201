# 함수의 응용

---

## **1. 여러 개의 결과값 반환하기**

{: .warning }
**함수의 반환값은 언제나 하나입니다!**

Python에서 함수는 기술적으로 단 하나의 객체만을 반환할 수 있습니다. 하지만 return 문 뒤에 여러 값을 쉼표(,)로 구분하여 나열하면, Python은 이를 자동으로 하나의 **튜플(Tuple)**로 묶어 처리합니다. 따라서 우리 눈에는 여러 개가 반환되는 것처럼 보이지만, 실제로는 '튜플'이라는 하나의 보따리가 전달되는 것입니다.

```python 
def add_and_mul(a, b):
    return a + b, a * b  # (a+b, a*b) 형태의 튜플 하나를 반환

# 변수 하나로 받으면 튜플로 저장됨
result = add_and_mul(3, 4)
print(result)  # (7, 12)
```

### **📦튜플 언패킹 (Tuple Unpacking)의 활용**
반환된 결과값들을 각각 독립된 변수에 담아 사용하고 싶을 때는 언패킹(Unpacking) 기법을 활용합니다. 함수의 결과값 개수와 동일하게 변수를 쉼표로 나열하여 호출하면 됩니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
# 반환된 튜플을 분리하여 각각 저장
res1, res2 = add_and_mul(3, 4)

print(res1)  # 7
print(res2)  # 12
```

### **💡 Note: 다양한 자료형의 혼합 반환**
반환값에는 정수나 실수뿐만 아니라 리스트와 같은 복합 자료형도 포함될 수 있습니다. 이 경우에도 전체 결과는 하나의 튜플로 묶이며, 언패킹 시 각 변수는 해당 자료형을 그대로 유지합니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
def my_trig_sum(a, b):
    out1 = a + b
    out2 = a - b
    return out1, out2, [out1, out2] # 숫자 2개와 리스트 1개를 반환

c, d, e = my_trig_sum(2, 3) #
# e 에는 [5, -1] 이라는 리스트가 담기게 됩니다.
```

## **2. 매개변수 초깃값 설정과 지정 호출**

### **매개변수 초깃값 (Default Value)**
함수를 정의할 때 매개변수에 미리 값을 대입해 두면, 호출 시 해당 인자를 생략해도 설정된 초깃값이 자동으로 적용됩니다. 특정 값이 반복적으로 사용되는 경우 코드를 간결하게 만들어 줍니다.

### **매개변수 이름을 지정하여 호출하기**
함수를 호출할 때 매개변수의 이름을 직접 지정하여 값을 전달할 수 있습니다. 이 방식을 사용하면 정의된 순서와 상관없이 원하는 매개변수에 값을 정확히 전달할 수 있어 가독성이 향상됩니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
# day와 name에 초깃값 설정 
def print_greeting(name='Student', day='Monday'):
    print(f"Greetings, {name}, today is {day}")

# 1. 인자 없이 호출 (모든 초깃값 사용) 
print_greeting() # Greetings, Student, today is Monday

# 2. 특정 매개변수만 지정하여 호출 
print_greeting(name='Alex') # Greetings, Alex, today is Monday

# 3. 매개변수 이름을 지정하여 순서 변경 
print_greeting(day='Friday', name='Timmy') # Greetings, Timmy, today is Friday
```

{: .warning }
**주의: 매개변수의 배치 순서**  
초깃값이 설정된 매개변수는 반드시 초깃값이 없는 매개변수보다 뒤쪽에 배치해야 합니다. 만약 def func(name='User', age):와 같이 작성하면, 호출 시 입력된 값이 어느 변수에 대입될지 판단할 수 없어 SyntaxError가 발생합니다.

## **3. 가변 매개변수 (*args)**

### **입력 인자의 개수가 유동적일 때**
매개변수 이름 앞에 `*`을 붙이면, 호출 시 전달되는 여러 개의 인자를 하나의 **튜플(Tuple)**로 묶어서 받습니다. 이를 통해 인자의 개수가 확정되지 않은 상태에서도 유연하게 데이터를 처리할 수 있습니다.

### **🚀 TRY IT!**
{: .text-blue-200 }

```python
def add_all(*args):
    """전달된 모든 숫자의 합을 계산 (for 문 미사용)"""
    # args는 (1, 2, 3...) 형태의 튜플이므로 sum() 함수로 바로 계산 가능합니다.
    return sum(args)

# 다양한 개수의 인자로 호출
print(add_all(10, 20))           # 결과: 30
print(add_all(1, 2, 3, 4, 5))    # 결과: 15
```

### **💡핵심 포인트**

- **튜플(Tuple)의 성질**: *args는 입력된 값들을 하나의 보따리(튜플)에 담는 역할을 합니다.
- **함수와의 연결**: 파이썬의 sum(), max(), min(), len()과 같은 내장 함수들은 튜플이나 리스트를 통째로 받아 처리할 수 있으므로, for 문 없이도 강력한 기능을 구현할 수 있습니다.

### **고정 매개변수와 가변 매개변수 혼합하기**
고정된 개수의 매개변수와 개수가 변하는 `*args`를 함께 사용할 수 있습니다. 이 경우, **고정 매개변수를 반드시 앞쪽에 배치해야 합니다**. 파이썬은 앞에서부터 차례대로 값을 대입한 뒤, 남는 값들을 모두 `args` 튜플에 담기 때문입니다.

### **🚀 TRY IT!**
{: .text-blue-200 }
예시 코드: 연산 종류 지정하기

```python
def calculate(operation, *args):
    """첫 번째 인자로 연산 종류를 정하고, 나머지는 연산할 데이터들로 받음"""
    if operation == "sum":
        return sum(args)
    elif operation == "count":
        return len(args)

# 첫 번째 인자는 operation에, 나머지는 args에 담깁니다.
print(calculate("sum", 1, 2, 3, 4, 5))   # 결과: 15
print(calculate("count", 1, 2, 3, 4, 5)) # 결과: 5
```

## **4. 변수의 유효 범위 - 지역변수와 전역변수**
변수는 선언된 위치에 따라 그 값이 유효한 범위가 결정됩니다.

### **📍 지역 변수 (Local Variable)**
함수 **내부**에서 정의된 변수입니다.
- **유효 범위**: 해당 함수가 실행되는 동안에만 살아있으며, 함수가 종료되면 메모리에서 사라집니다.
- **특징**: 함수 외부에서는 이 변수에 접근할 수 없습니다.

### **🌐 전역 변수 (Global Variable)**
함수 **외부**(메인 프로그램)에서 정의된 변수입니다.
- **유효 범위**: 프로그램 전체에서 접근이 가능합니다.
- **특징**: 함수 내부에서도 읽을 수는 있지만, 함수 안에서 직접 수정하려면 별도의 처리가 필요합니다.
