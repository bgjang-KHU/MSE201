# Chapter 3. 함수 (Functions)

## 💡 왜 함수를 배워야 할까요? 

우리가 함수를 사용하는 결정적인 이유는 두 가지입니다.

1. 데이터 처리의 자동화와 재사용성
동일한 계산이나 데이터 처리 과정을 반복할 때마다 코드를 재작성하는 것은 비효율적이며 휴먼 에러의 원인이 됩니다. 
반복되는 로직을 함수로 정의해두면, 다양한 입력값에 대해 동일한 규칙을 즉각 적용하는 자동화된 연산 도구로 활용할 수 있습니다.

    - 예시: sin(x) 연산의 내부 구현을 매번 작성하지 않고 `math.sin()` 함수를 호출하여 결과를 얻는 것과 같습니다.

2. 프로그램의 구조화
복잡한 프로그램을 기능별 단위로 분리하면 전체적인 흐름을 논리적으로 파악할 수 있습니다. 마치 실험실의 개별 공정처럼 프로그램을 구조화하면, 코드의 가독성이 높아질 뿐만 아니라 오류 발생 시 특정 모듈만 점검하면 되므로 유지보수가 매우 용이해집니다.
