# 함수의 기본 (Functions Basics)

---

## **함수의 정의 (Defining a Function)**

 Python에서 나만의 함수를 만드는 가장 일반적인 방법은 `def` 키워드를 사용하는 것입니다. 함수는 크게 정보를 받는 **머리(Header)**와 실제 일을 하는 **몸통(Body)**으로 나뉩니다.

### **🛠️ 함수의 기본 구조**

```python
def 함수이름(매개변수1, 매개변수2, ...): # (1) 함수 헤더 (Function Header)
    '''
    함수에 대한 설명 (Descriptive String)
    '''
    # (2) 함수 내용 (Function Body) 
    Function statement (실행문)
    
    return 결과 # (3) 반환값 (선택 사항)
```

### **🔍 함수의 핵심 구성 요소**

**1. 함수 헤더 (Function Header)**
- 반드시 `def` 키워드로 시작합니다.
- 함수 이름 뒤에는 소괄호 `()`와 매개변수 (입력값)을 적습니다.
- 문장의 끝에는 반드시 **콜론(`:`)** 붙입니다.

**2. 함수 내용 (Function Body)**
- 헤더 아래 **들여쓰기 (Indentation)**된 블록을 의미합니다. (보통 공백 4칸을 권장합니다.)
- **설명 문자열 (Descriptive string)**: 함수가 무슨 일을 하는지 적어두는 곳입니다. 없어도 무방하지만 나중에 `help(함수이름)` 이 명령어로 내용을 확인할 수 있어 다른 사람들과 협업에는 필수적입니다. 
- **실행문 (Function statement)**: 함수가 호출되었을 때 실행되는 단계별 지시 사항입니다. `#`으로 시작하는 주석은 실행되지 않으며 코드 이해를 돕기 위해 작성합니다.

**3. 반환문 (Return Statement)**
- 함수가 모든 일을 마친 후 최종 결과값을 함수 밖으로 돌려주는 명령입니다.
- **선택 사항**이며, 돌려줄 값이 없다면 생략할 수 있습니다. 정수, 리스트, 심지어 다른 함수까지 모든 데이터 타입을 반환할 수 있습니다.

## **📂 함수의 형태 비교**

### **1. 입력(매개변수) ❌, 반환값 ❌**
가장 단순한 형태로, 호출하면 정해진 일을 수행하고 끝납니다. 주로 반복적인 안내 메시지를 출력할 때 사용합니다.

```python
def say_hello():
    """인사와 함께 학과명을 출력하는 함수"""
    print("Hello MSE")

# 호출
say_hello()
```

### **2. 입력(매개변수) ❌, 반환값 ⭕**
함수 내부에서 데이터를 생성하거나 입력받아 밖으로 던져주는 형태입니다. 외부에서 결과값을 받아서 다른 계산에 활용할 때 유용합니다.

```python
def get_user_number():
    """사용자에게 숫자를 입력받아 반환하는 함수"""
    num = float(input("분석할 데이터를 입력하세요: "))
    return num # 결과값을 함수 밖으로 보냄

# 호출 및 활용
value = get_user_number()
print(f"입력된 값은 {value}입니다.")
```

### **3. 입력(매개변수) ⭕, 반환값 ❌**
입력값는 받지만, 결과를 돌려주지 않고 함수 안에서 소비하는 형태입니다. 특정 데이터를 가공해서 바로 화면에 보여줄 때 자주 씁니다.

```python
def greet_student(name):
    """이름을 입력받아 환영 인사를 출력하는 함수"""
    print(f"신소재공학과 {name} 학생, 반갑습니다!")

# 호출 (인수 '홍길동'을 전달)
greet_student("홍길동")
```

### **4. 입력(매개변수) ⭕, 반환값 ⭕**
전형적인 함수의 모습입니다. 입력값을 받아 실행문에 따른 작업을 하고 결과를 반환합니다.

```python
def calculate_square(x):
    """입력받은 숫자의 제곱을 계산하여 반환하는 함수"""
    result = x ** 2
    return result

# 호출 및 활용
answer = calculate_square(5)
print(f"5의 제곱은 {answer}입니다.")
```

## **함수의 호출과 결과값 활용**

### **📞 함수의 호출**
함수는 정의(`def`)하는 것만으로는 실행되지 않습니다. 함수의 이름을 소괄호 ()와 함께 불러주는 호출(Call) 과정이 있어야 실제 로직이 가동됩니다. 매개변수 (argument)가 있는 경우는 정의된 형식에 맞춰 필요한 인자를 전달해야 합니다.

{: .warning }
**함수 호출 시 소괄호`()` 사용 주의**
정의된 입력 인자 (Argument)가 없는 함수라 하더라도, 함수를 실행하기 위한 호출 시에는 반드시 이름 뒤에 소괄호`()`를 붙여야 합니다. 소괄호를 생략하면 함수가 실행되지 않고 함수 객체 자체를 가리키게 되므로 작성 시 유의하시기 바랍니다.


### **📥 반환값의 저장**

`return` 문을 통해 도출된 결과값은 함수가 종료된 후 호출된 위치로 되돌아옵니다. 이 값을 변수에 대입하여 저장해야만 이후의 연산에서 재사용할 수 있습니다.

### **💡 데이터 처리의 관점: Print vs Return**
- `print()`: 결과를 단순히 콘솔 창에 출력할 뿐, 데이터로서 메모리에 저장되지는 않습니다.
- `return`: 함수의 결과 데이터를 외부 변수로 전달하여, 다른 수식이나 함수의 입력값으로 사용할 수 있게 합니다.

```python
def get_melting_point():
    return 1538

# 반환값을 변수에 저장하여 활용
iron_mp = get_melting_point()
print(iron_mp + 100) # 변수에 저장되었으므로 연산 가능
```
